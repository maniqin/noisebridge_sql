                  ## SELECT statement
                  - SELECT, as the word implies, "selects" data from tables
                  - SQL way to retrieve data from one or multiple tables
                  - May be the most common statement you'll see around
                  - 99.9% of the times it will be followed by a FROM
                  - Doesn't matter if typed upper or lower case
                  - Indentation and spacing in SQL doesn't matter

SELECT from a table

  • All columns 
                      
                    SELECT * FROM table_name
  • Specific columns, choose the order you want 
                      
                    SELECT  
                      column_name_1,
                      column_name_2,
                      column_name_3
                    FROM table

What if the table outputs too many rows?

Use limit!
              
                SELECT 
                  column_1,
                  column_2 
                FROM table_name
                LIMIT 100;
